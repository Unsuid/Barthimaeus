# Barthimaeus Rescue Repo

Private recovery repository for Barthimaeus, Daniel's OpenClaw agent persona and working setup.

This is not a raw backup of `/home/deck/.openclaw`. It is a curated restore kit: identity, operating notes, avatar assets, and safe project briefs that help rebuild the assistant on a new machine if the Steam Deck fails.

## Contents

- `identity/` - Core persona and safe user context.
- `memory/` - Curated memory seeds, not raw logs.
- `avatars/` - Profile and Gateway avatar assets.
- `docs/` - Restore notes, project briefs, and OpenClaw integration notes.
- `tools/` - Small safe helpers, if any are added later.

## Ground Rules

- No `.env` files, tokens, API keys, passwords, cookies, or private profiles.
- No raw emails, chats, browser exports, session logs, or full workspace dumps.
- No Home Assistant, n8n, TV, SSH, or device configs containing secrets.
- Prefer short curated notes over copying local state.

This repo is the lifeboat, not the attic.
