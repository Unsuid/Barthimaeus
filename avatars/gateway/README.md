# Barthimaeus Gateway Avatar

Local animated avatar concept for the OpenClaw Gateway.

- `barthimaeus-gateway.html` is a self-contained HTML/SVG component.
- `?state=idle`, `?state=sleep`, `?state=thinking`, `?state=working`, `?state=irked`, and `?state=proud` select the state on load.
- The component uses CSS animations only, with `prefers-reduced-motion` support.
- `barthimaeus-gateway-v2.png` is the current preview render.

Design direction: stylized smoky djinn, ruby/gold face, teal thinking accent, compact enough for a Gateway panel without going full fantasy-poster.
